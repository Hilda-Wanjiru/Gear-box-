I. Introduction to Embedded systems

An embedded system is a combination of computer hardware and software designed for a specific function.


Embedded systems typically contain a microprocessor -- or a microcontroller-based system, memory and input/output devices


II. Introduction to IDE's

IDE definition: An integrated development environment (IDE) is a software application that helps programmers develop software code efficiently. It increases developer productivity by combining capabilities such as software editing, building, testing, and packaging in an easy-to-use application.


Types of IDE's

1. Thonny IDE

Thonny is an Integrated Development Environment (IDE) for Python that aims to make learning and teaching programming easier. Developed by Aivar Annamaa at the University of Tartu, Thonny is designed to provide a beginner-friendly environment for writing and debugging Python code. It is open-source software and is available for Windows, macOS, and Linux platforms.

2. Arduino IDE

Arduino Integrated Development Environment (IDE) is an open source IDE that allows users to write code and upload it to any Arduino board. Arduino IDE is written in Java and is compatible with Windows, macOS and Linux operating systems.

III. Introduction to C and C++

C definition

C is a general-purpose programming language created by Dennis Ritchie at the Bell Laboratories in 1972. It is a very popular language, despite being old. The main reason for its popularity is because it is a fundamental language in the field of computer science

C variables

Variables are containers for storing data values, like numbers and characters.
In C, there are different types of variables (defined with different keywords), for example:
a. int - stores integers (whole numbers), without decimals, such as 123 or -123
b. float- stores floating point numbers, with decimals, such as 19.99 or -19.99
c. char - stores single characters, such as 'a' or 'B'. Characters are surrounded by single quotes
